I started my career building desktop applications with **Visual Basic** and **C#**. Those early years taught me the fundamentals of software architecture, data modeling, and the importance of writing clean, maintainable code.

## The Shift to Web

As the industry moved toward web-based solutions, I transitioned into full-stack web development. I picked up **PHP** and **JavaScript**, then deepened my expertise with frameworks like **Laravel** and **CodeIgniter**.

One of my first major web projects was a financial intelligence system for Bufalinda (BBI). It integrated with Microsoft Dynamics SL ERP to collect and transform data across purchasing, sales, inventory, and accounting departments. This project taught me how to work with multiple MSSQL databases and build complex reporting systems.

## Expanding My Toolkit

Over time I added more tools to my belt:

- **Frontend**: React, jQuery, HTML5, CSS3, Bootstrap
- **Backend**: Laravel, CodeIgniter, Node.js, C# MVC
- **Databases**: MySQL, Microsoft SQL Server
- **Tools**: Git, IIS, ERP integration

## What I'm Focused On Now

I'm currently deepening my React skills and exploring modern frontend architectures. I believe in being a lifelong learner — technology evolves fast and staying curious is the only way to keep up.
